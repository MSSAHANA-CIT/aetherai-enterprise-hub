# AetherAI Enterprise Hub — Frontend

Premium React frontend for the AetherAI Enterprise Hub platform.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Lucide React

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

App: http://localhost:5173

## Structure

- `src/pages/` — Page components
- `src/components/ui/` — Reusable UI primitives
- `src/components/landing/` — Landing page sections
- `src/layouts/` — Layout wrappers
- `src/routes/` — React Router configuration
- `src/lib/` — Utilities

## Build

```bash
npm run build
npm run preview
```
